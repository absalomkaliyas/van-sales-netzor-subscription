# Public Assets

## Logo

Place your NETZOR logo file here as `logo.png`.

The logo should be:
- Format: PNG (with transparency preferred)
- Size: At least 200x200px (will be displayed at 40x40px)
- File name: `logo.png`

The logo will be displayed in:
- Dashboard navigation
- Login page (optional)

If the logo file is not found, the image will be hidden gracefully and only the text will be displayed.
