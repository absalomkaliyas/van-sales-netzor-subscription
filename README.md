# VAN Sales Netzor Subscription

## Project Overview

This project is set up with Git version control for cloud synchronization.

## Getting Started

### Prerequisites

- Git installed on your system

### Cloud Sync Setup

This repository is initialized with Git. To enable cloud sync, connect it to a remote repository:

#### Option 1: GitHub
1. Create a new repository on GitHub
2. Run the following commands:
   ```bash
   git remote add origin https://github.com/yourusername/your-repo-name.git
   git branch -M main
   git push -u origin main
   ```

#### Option 2: GitLab
1. Create a new project on GitLab
2. Run the following commands:
   ```bash
   git remote add origin https://gitlab.com/yourusername/your-repo-name.git
   git branch -M main
   git push -u origin main
   ```

#### Option 3: Other Git Hosting Services
Follow similar steps for other Git hosting services (Bitbucket, Azure DevOps, etc.)

### Basic Git Commands

- `git status` - Check the status of your repository
- `git add .` - Stage all changes
- `git commit -m "Your message"` - Commit changes
- `git push` - Push changes to remote repository
- `git pull` - Pull latest changes from remote repository
