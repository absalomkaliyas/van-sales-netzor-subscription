# Expo SDK Compatibility Table

| Expo SDK | React Native | Gradle Version | AGP Version | Kotlin Version |
|----------|--------------|----------------|-------------|----------------|
| **52** (Latest) | 0.76.x       | 8.8+           | 8.5.x       | 1.9.24         |
| **51** (Stable) | 0.74.x       | 8.3 - 8.6      | 8.2.x       | 1.9.23         |
| **50** (Legacy) | 0.73.x       | 8.0 - 8.2      | 8.1.x       | 1.8.10         |
| **49** (Legacy) | 0.72.x       | 7.5 - 8.0      | 7.4.x       | 1.8.0          |

**Note**: To use Gradle 8.3+, you MUST use **Expo SDK 51** or higher. SDK 50 modules are often incompatible with AGP 8.2+.
